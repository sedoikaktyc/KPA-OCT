1.5 в PuTTY Key Generator выбрать сохранить приватный ключь

5. в папке где будет распологаться проект сделать git clone с urll удаленного репозитория

6. Настройка Merge Tool Unity 
6.1 TortoiseGit -> Settings -> Merge Tool -> Advanced -> add  Добавить файл с расшырение .unity и указать путь
C:\Program Files\Unity\Hub\Editor\2019.4.15f1\Editor\Data\Tools\UnityYAMLMerge.exe merge -p %base %theirs %mine %merged
6.2 повторить для расширении .prefab 
6.3 В файле C:\Program Files\Unity\Hub\Editor\2019.4.15f1\Editor\Data\Tools\mergespecfile добавить после 23 строки 
# Tortoise Git Merge
* use "%programs%\TortoiseGit\bin\TortoiseGitMerge.exe" -base:"%b" -mine:"%l" -theirs:"%r" -merged:"%d"

7. запусить проект через unity hub\
